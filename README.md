Cygen assignment

Demo
https://cygen-task.vercel.app/

Tech Stack
Client: ReactJs,Material-ui

Run Locally
Clone the project

```
git clone https://github.com/ravikrishnudu/cygen-task.git
```

Go to the project directory

```
cd cygen-task
```

Install dependencies

```
yarn
```

Start the server

```
 yarn start
```
