# Cygen assignment

## Demo

https://cygen-assignment.vercel.app/

## Tech Stack

**Client:** ReactJs,Material-ui

## Run Locally

Clone the project

```
git clone https://github.com/ravikrishnudu/cygen-assignment.git
```

Go to the project directory

```
cd cygen-assignment


```

Install dependencies

```
yarn
```

Start the server

```
yarn start
```
